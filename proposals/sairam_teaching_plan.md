## ðŸ›  Teaching Strategy (The "Impact" Methodology)

1. **The "20-Line" Rule**: Every project shown to Sairam students is under 20 lines of code. This reduces "syntax anxiety" and focuses on **Architectural Logic**.
2. **Predict-Observe-Correct**: Before running each script (e.g., `05_cycle.py`), the class predicts the graph's path, observes the terminal output, and discusses the state transitions.
3. **Template-First Development**: Students start with the pre-configured `.venv` and `requirements.txt` to ensure 100% of their time is spent on code logic, not environment debugging.
